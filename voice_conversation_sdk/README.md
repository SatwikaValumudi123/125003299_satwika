# Voice Conversation SDK

## Overview
This SDK allows the creation of voice bots powered by Large Language Models (LLMs) with minimal latency. It converts speech to text (STT), queries an LLM, converts the response to speech (TTS), and streams the audio back to the user.

## Requirements
- Python 3.7+
- Deepgram API Key
- OpenAI API Key
